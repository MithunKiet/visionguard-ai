# VisionGuard AI

VisionGuard AI is an open-source factory safety monitoring platform designed to improve workplace safety through real-time computer vision.

## Features

- Helmet Detection
- Safety Vest Detection
- Person Counting
- Zone Occupancy Monitoring
- Overcrowding Alerts
- Real-Time Dashboard
- Snapshot-Based Violation Logging
- Supervisor Notifications
- Multi-Camera Support

## Technology Stack

- Python
- FastAPI
- OpenCV
- RT-DETR
- PostgreSQL
- Docker

## Use Cases

- Factory Safety Compliance
- PPE Monitoring
- Restricted Area Monitoring
- Industrial Workforce Safety
- Smart Manufacturing
